# Carte Grise Document - QR Code Access

This repository hosts a QR-code accessible PDF document for Carte Grise.

## How it works
1. PDF is hosted externally (Google Drive/DocDroid)
2. HTML page embeds the PDF viewer
3. QR code points to this GitHub Pages site
4. Mobile-optimized interface

## Setup
1. Replace `PDF_URL` in index.html
2. Enable GitHub Pages in Settings
3. Generate QR code for your GitHub Pages URL
